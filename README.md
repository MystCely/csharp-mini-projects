# C# Mini Projects
A collection of small C# practice projects.

## Topics Covered

- Console input and output
- Conditional logic and branching
- Loops and repetition
- Random number generation
- Basic game mechanics
- Simple state and score tracking
- Object-oriented programming (classes and objects)
- Methods and method design
- Lists and collections
- Encapsulation
- Constructors
- Object relationships
- Menu-based console navigation


## Purpose
The goal of this repository is to apply C# fundamentals in small, focused mini-projects.
